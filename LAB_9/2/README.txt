Что реализовано:

Похожий у разработанного приложения интерфейс на обычный mc в Ubuntu; // Далее разработанное приложение будет именоваться mc

Автоматический подгон соотношения сторон, даже при изменении масштаба консоли интерфейс будет адаптироваться;

Вывод директорий и файлов;

При использовании обычных символов они(символы) будут отображаться в 4 окне (нижнее окно с подобием командной строки); // Далее подобие командной строки - командная строка

Полная поддержка bash команд;

При нажатии Enter по директории произойдёт переход в директорию;

При нажатии Enter по файлу(не исполняемому), и, если ранее не были использованы обычные символы, произойдёт переход в nano, при нажатии Enter по исполняемому файлу произойдёт его выполнение в консоли, после произойдёт переход обратно к mc;

При раннем введение обычных символов и нажатии Enter произойдёт выполнение bash команды, командная строка будет очищена;

При нажатии TAB произойдёт перемещение на противоположное окно;

При нажатии Backspace произойдёт очистка предыдущего символа;

При использовании стрелок вверх / вниз произойдёт перемещение по директориям или файлам, текущее положение указателя на файл или директорию окрашивается;

При использовании стрелок влево / вправо произойдёт перемещение в командной строке;

При введении слова "exit" произойдёт завершение программы mc;
